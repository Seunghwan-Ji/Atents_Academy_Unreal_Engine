##언리얼 엔진 설치법
1. 구글에 "언리얼 엔진" 검색 후 공식 홈페이지 접속.
2. 우측 상단 "다운로드" 버튼 클릭 및 "런처 다운로드하기" 버튼 클릭 후 에픽게임즈 런처 설치 진행.
3. 에픽게임즈 런처 실행 및 로그인, 왼쪽 메뉴바에서 "언리얼 엔진" 선택 후 탭에서 "라이브러리"바 선택.
4. 엔진 버전 "+" 버튼 클릭 후 원하는 버전 선택.
5. 설치 옵션에서 "디버깅을 위한 편집기 기호" 체크.
6. 윈도우 환경에서만 개발할 경우 "타겟 플랫폼" 리스트 모두 체크 해제. (용량 확보 장점)

##언리얼 엔진 체험해보기
1. 에픽게임즈 런처 실행 -> 언리얼 엔진 -> 라이브러리 -> 엔진 실행.
2. 게임 메뉴 선택 -> "일인칭", "삼인칭", "내려보기" 등 하나 선택 후 생성.
3. 기본 빈 프로젝트: 게임 -> "기본" 생성.
- 이후엔 런처를 통해서가 아닌, 로컬에서 프로젝트 폴더 내 프로젝트 실행 파일로 엔진 실행 가능.

##에셋 둘러보기
1. 에픽게임즈 런처 실행 -> 언리얼 엔진 -> 라이브러리 -> 펩 라이브러리 항목에서 "Fab.com을 방문하세요" 클릭해서 접속.
2. 홈페이지에서 로그인 후 우측 상단 구 모양 버튼 눌러서 언어 변경.
3. 홈페이지 오른쪽에 추천 콘텐츠 "기간 한정 무료" 메뉴 클릭.
4. 무료 에셋들을 모두 장바구니에 담고, 장바구니에서 결제 -> "PLACE ORDER" 클릭.
5. 에픽게임즈 런처의 펩 라이브러리 항목에서 새로고침 버튼을 눌러 에셋들이 들어왔는지 확인.
