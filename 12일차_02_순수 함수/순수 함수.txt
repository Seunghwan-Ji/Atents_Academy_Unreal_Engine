- 순수 함수는 기본 함수와 달리 실행핀이 없음.
- 기본 함수는 실행핀이 통과할 때 딱 한번만 실행되고 리턴값을 저장해놓음.
- 기본 함수는 자신의 리턴값 핀을 여러 함수 노드에서 사용해도 한 번만 실행됨.
- 순수 함수는 실행핀이 없기 때문에 언제 실행되는지 알 수 없음.
- 따라서 순수 함수는 자신의 출력값을 사용하는 비순수(기본 함수) 함수에 의존하여 실행하게됨.
- 실행 횟수는 자신의 출력값을 사용하는 비순수 함수 노드의 개수와 같음.
- 단, 하나의 비순수 함수 노드에서 순수 함수를 여러번 호출할 수 없음. (순수 함수의 출력핀을 비순수 함수의 여러 입력핀에 연결한 경우)
- 내 블루프린트 -> 함수에서 특정 함수를 진입해 함수 시작 노드를 선택하고 디테일 -> 퓨어를 체크하면 순수 함수로 만들 수 있음.
- 순수 함수의 조건:
	- 1. 같은 입력값에 대해 항상 같은 출력값이 나와야함.
	- 2. 자신의 함수 내부에서 외부의 값을 변경하지 않아야함.
	- 하지만 언리얼은 이 조건들을 충족하지 못해도 퓨어 체크를 가능하게 했지만, 실제로 조건 충족이 안된 순수 함수 사용시 오류가 발생하므로 주의 필요.

- BP_Character -> SelectWeapon -> Get State Montage Data 함수 노드는 항상 같은 입력에 대해 같은 출력을 하며, 외부 값을 건드리지 않으므로 퓨어 함수로 바꿔줘도 문제 없음.
- Get State Montage Data 노드 더블클릭 -> GetStateMontageData 노드 선택 -> 디테일 -> 퓨어 체크.
- 다음으로 BP_Weapon -> 함수 -> GetStateMontageData -> Get Montage Data 노드도 퓨어 함수 자격이 있음.
- Get Montage Data 노드 더블클릭 -> GetMontageData 노드 선택 -> 디테일 -> 퓨어 체크.
