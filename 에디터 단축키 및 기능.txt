## 단축키
Ctrl + Tab: 탭 전환. (위에서 아래로 순회)
Ctrl + Shift + Tab: 탭 전환. (아래에서 위로 순회)
Ctrl + Shift + S: 에디터에서 작업중인 모든 내용 저장.
Ctrl + Space: 콘텐츠 드로어 열기.
Shift + F11: 에디터 창의 사이즈가 작업 표시줄 영역까지 확장됨.
Ctrl + Z: 작업 실행 취소.
Ctrl + Y: 취소한 작업 다시 실행.

### 뷰포트 관련
Q: 오브젝트 단순 선택.
- ESC: 선택된 오브젝트 해제.
W: 선택된 오브젝트 이동.
- 직각 변: 두 축 단위로 이동.
- 흰색 구: 세 축 단위로 이동.
E: 선택된 오브젝트 회전.
R: 선택된 오브젝트 스케일 조정.
Ctrl + `: 선택된 오브젝트의 좌표계를 월드 또는 오브젝트 기준으로 변경.
End: 선택된 오브젝트를 지면에 접하도록 착지.
ESC: 선택된 오브젝트 선택 해제.
F: 선택된 오브젝트 위치로 날아감.
Shift + F1: 실행중인 뷰포트에서 마우스 포인터가 생김.
F8: 실행중인 뷰포트에서 에디터를 조작할 수 있음.
- 중지 버튼 오른쪽에 있는 버튼의 단축키.

###블루프린트 관련
F7: 컴파일.
Ctrl + D: 선택된 노드 복제.
Shift + delete: 간선에 붙은 점을 선택했을 때 제거할 수 있음.
P + 좌클릭: 이벤트 그래프에 BeginPlay 노드 생성.
Ctrl + A: 모든 노드 선택.
Ctrl + 드래그: 노드가 선택된 채로 추가적으로 다른 노드을을 더 선택.
노드 정렬 단축키: 노드 선택 -> 정렬 -> 단축키 조합 확인.
C + 좌클릭: 코멘트 생성.
S + 좌클릭: 시퀀스 노드 생성.

## 기능
- 프로젝트를 구성하는 폴더들 중 DerivedDataCache, Intermediate, Saved 이 세 폴더는 보통 백업할 필요 없음. 왜냐하면 전부 자동 생성 / 임시 데이터들이라서, 프로젝트 열면 다시 만들어지기때문.
- 좌측 상단 편집 -> 에디터 개인설정 -> 지역&언어 -> 에디터 언어 -> 한국어로 언어 설정 가능.
- 우측 상단 세팅 -> 엔진 퀄리티 세팅 -> 자동 또는 수동으로 엔진 그래픽 퀄리티 조절 가능.
	- "하드웨어 퍼포먼스를 모니터링할까요?" 부분은 체크하면 상황에 따라서 퀄리티가 유동적으로 조절됨.
- 모든 UI에서 파란색 모서리 부분은, 누르면 탭이 생김. 탭이 하나인 경우 탭을 우클릭하여 탭 숨김 가능.
- 탭 우클릭 -> 사이드바에 고정을 클릭하면 탭이 우측 사이드로 이동하여 고정됨.
- 아웃라이너 탭 밑에 디테일 탭에서 현재 선택된 오브젝트의 세부사항들을 볼 수 있고, 수치들을 조절할 수 있음.
- 좌측 상단 창 메뉴에서 에디터 UI 배치도를 저장, 불러오기 등을 할 수 있음.
- 좌측 상단 편집 -> 프로젝트 세팅 -> 맵&모드 -> 에디터 시작맵에서 매번 에디터 실행시 시작할 맵을 설정할 수 있음.
- 좌측 상단 편집 -> 에디터 개인설정 -> 외형 ->에셋 에디터 실행 위치를 Main Window로 해주면 에디터 내에서 새로운 탭을 열 때 새 창에서 뜨지 않고 기존 창 탭 배열에 추가됨.
- 디테일 창에서 우측 톱니바퀴 -> 모든 카테고리 접기/펴기 가능.

### 뷰포트 관련
- 뷰포트 우측 상단 세팅 -> 엔진 퀄리티 세팅에서 그래픽 성능을 조절할 수 있음.
- 뷰포트를 우클릭한 채로 마우스를 움직이면 시야를 움직일 수 있음.
- 뷰포트를 우클릭한 채로 W, A, S, D 키를 누르면 맵 내에서 이동할 수 있음.
- 뷰포트를 우클릭한 채로 Q, E 키를 누르면 맵 내에서 상하로 이동할 수 있음.
- 뷰포트에 마우스를 대고 마우스 휠을 움직이면 시야를 확대 및 축소할 수 있음.
- 뷰포트 내에서 좌클릭으로 오브젝트를 선택할 수 있음.
- 뷰포트 우측 상단에서 그리드 배열 아이콘을 누르면 선택된 오브젝트를 그리드 단위 만큼 이동시키거나, 그냥 부드럽게 이동시킬 수 있음.
- 뷰포트 우측 상단에서 각도 아이콘을 누르면 선택된 오브젝트를 특정 단위 만큼 회전시키거나, 그냥 부드럽게 회전시킬 수 있음.
- 뷰포트 우측 상단에서 오른쪽 위 방향 화살표 아이콘을 누르면 선택된 오브젝트를 그리드 단위로 스케일을 조정하거나, 그냥 부드럽게 조정할 수 있음.
- 뷰포트 우측 상단에서 카메라 아이콘을 눌러 카메라 속도를 조절할 수 있음.
- 뷰포트 우측 아웃라이너 탭에서 월드상에 존재하는 모든 오브젝트 리스트를 볼 수 있음.
- 블루프린트 내 뷰포트에서 왼쪽 상단 "원근" 버튼을 누르면 다양한 차원에서 트랜스폼을 조정할 수 있음. (예: 충돌 영역 스케일 조정)
- 레벨 뷰포트에서 플레이 버튼 오른쪽에 점 3개 아이콘 -> 현재 카메라 위치를 선택하면 플레이 시 플레이어가 카메라 위치에서 스폰됨.

### 블루프린트 관련
- 블루프린트에서 플레이 버튼을 클릭하면 모든 클래스에 대해 컴파일함.
- Ctrl 키를 누른채로 핀을 클릭하면 다른 핀과 끊지 않고 간선을 옮길 수 있음.
- Alt 키를 누른채로 핀을 클릭하면 다른 핀과 연결을 끊을 수 있음.
- 내 블루프린트 탭에서 Ctrl 키를 누른채로 변수를 드래그하면 Get 노드를 배치할 수 있음.
- 내 블루프린트 탭에서 Alt 키를 누른채로 변수를 드래그하면 Set 노드를 배치할 수 있음.
- 값이 여러개인 핀을 우클릭 -> 구조체 핀 분할을 선택하면 핀을 쪼갤 수 있음.
- 또는 핀을 당겨서 Break Vector 노드 추가. (구조체 멤버에 접근을 하면서 동시에 분기를 생성해야되는 상황일 때 사용.)
- 쪼개진 핀중 하나를 우클릭 -> 구조체 핀 재결합을 선택하면 핀을 다시 합칠 수 있음.
- 마우스 휠을 움직이면 이벤트 그래프를 확대 및 축소 가능.
- Ctrl 키를 누른채로 휠을 움직이면 확대 최대치를 넘길 수 있음.
- 간선에 마우스를 대고 선이 굵어졌을 때 더블클릭하면 그 지점에 점(경유 노드)이 생기는데 이 점을 Ctrl 키를 누른채로 점을 드래그하면 간선을 움직일 수 있음. (간선 정리용)
- 또는 점 가장자리 부분에 마우스를 대고 포인터가 십자 화살표가돼면 드래그해서 간선을 움직일 수 있음.
- 에디터 개인설정 -> 외형 -> 머터리얼과 블루프린트 에디터에서 그리드 사용 체크 해제시 이벤트 그래프에서 그리드 배열 배경이 보이지 않게됨. (정리병 치료용)
- 에디터 개인설정 -> 외형 -> 그리드 스냅 크기를 1로 설정하면 노드나 간선 배치시 1단위로 움직임. (정리병 치료용)
- 핀 우클릭 -> 변수로 승격을 선택하면 그 변수 타입에 맞는 Set 노드가 생성됨.
- 노드에 마우스를 대고 있으면 노드의 좌측 상단에 말풍선이 생기는데, 클릭하면 주석을 달 수 있음.
- 또는 노드를 우클릭 -> 노드 코멘트 항목에서 주석을 작성해 달 수 있음.
- 에디터 개인설정 -> 블루프린트 에디터 세팅 -> 컴파일러 -> 컴파일 시 저장 -> On Success: 컴파일 성공 시 저장, Always: 항상 저장. (보통 Always)
- 에디터 개인설정 -> 블루프린트 에디터 세팅 -> 컴파일러 -> 노드 오류로 점프를 체크하면 컴파일 실패 시 원인이 되는 노드로 이동함.
- 입력 액션 클래스에서 트리거 발동 조건을 지정하지 않으면 기본적으로 "다운", 작동 한계치 0으로 설정됨.
- 애님 그래프에서 트랜지션 룰 버튼을 선택 -> 디테일 -> 트랜지션 룰 공유 -> 공유로 승격 -> 이름 짓기 -> 이름 예시: SharedRule
- 다른 트랜지션 룰 버튼 선택 -> 디테일 -> 트랜지션 룰 공유 -> 공유 사용 -> SharedRule
- 이러면 다른 트랜지션 룰에서 SharedRule 의 조건을 사용할 수 있음. (공통 조건을 가진 트랜지션 룰이 여러개인 경우 유지보수에 용이)
- 애님 그래프에서 우클릭 -> 스테이트 에일리어스 노드를 추가하면 이 노드는 다른 1개 이상의 노드를 대신할 수 있음. (간선이 너무 길게 이어질 경우 사용하기 용이)
- 어떤 노드를 대신할 지는, 추가한 스테이트 에일리어스 노드를 선택 -> 디테일 -> 스테이트 에일리어스 목록에서 대신할 노드 체크하면됨.
- 글로벌 에일리어스를 체크하면 모든 노드를 대신하며, 그 모든 노드가 항상 글로벌 에일리어스 노드의 트랜지션 룰의 조건을 따지게됨.
- 두 개 이상의 노드를 선택한 다음 아무 노드에 우클릭 -> 선택에서 코멘트 생성을 선택하면 그 노드들을 감싸서 주석을 달 수 있음.
- 또는 하나 이상의 노드를 선택한 후 'C' 키를 누르면 주석을 달 수 있음.
- 빈 공간을 클릭하고 'C' 키를 누르면 그 빈 공간에 주석을 달 수 있음.
- 노드를 선택하고 Shift + delete 키를 누르면 이 노드를 거쳐서 연결된 다른 노드들이 끊기지 않은 채로 삭제됨.
- 편집 -> 미사용 변수 삭제를 선택하면 정의된 변수들중 모든 곳을 통틀어서 사용되지 않는 변수 목록이 뜨고, 선택해서 지울 수 있음.
- B 키를 누른 채로 그래프 빈 공간을 클릭하면 Branch 노드를 생성할 수 있음.
- 노드를 선택하고 F9 키를 누르면 디버깅에 흔히 사용하는 중단점을 표시할 수 있음.
- 블루프린트 창에서 플레이 버튼을 누르고, 플레이 버튼 오른쪽에 선택된 디버그 오브젝트 없음 -> 현재 블루프린트 클래스 이름 선택 (예: BP_Character)
- 중단점을 찍은 노드까지 가는 상황을 진행하면 플레이가 중단되면서 중단점 노드로 진입하게됨.
- 그 상태에서 F10 키를 누르면 다음 노드로 넘어가고, F11 키를 눌렀을 때는 현재 노드가 함수인 경우 그 함수 안으로 들어감.
- Alt + Shift + F11 은 들어간 함수를 빠져나옴.
- Alt + Shift + F10 키 조합을 누르면 플레이를 이어서 진행함.
- 디버깅중 핀에 마우스를 갖다대면 들어온 값을 확인할 수 있음.
- 내 블루프린트 -> 변수 -> 변수에서 타입에 마우스를 갖다대고 좌클릭하면 타입을 배열 등으로 바꿀 수 있음.
- 경유 노드를 당겨서 노드 검색 -> drer -> Add Reroute Node 를 선택하면 경유 노드를 잇는 또 다른 경유 노드를 생성할 수 있음.
- 노드 두 개를 선택하고 Q 키를 누르면 먼서 선택한 노드를 후에 선택한 노드를 기준으로 정돈함.
- 내 블루프린트 -> 그래프 -> 추가 버튼을 누르면 이벤트 그래프가 추가되는데, 이 그래프는 기존 그래프에서 이어지는 그래프임. (기존 그래프에서 공간을 많이 사용했을 경우 유용.)
- 특정 노드를 선택하고 상단에 "관련없음 숨김" 버튼을 누르면 그 노드와 연관된 흐름만 보여줌.

### 애니메이션 에디터 관련
- Shift 키를 누른 채로 애니메이션 에셋을 더블클릭하면 새 탭에서 그 에셋을 볼 수 있음.

### 콘텐츠 드로어 관련
- 항목을 여러개 선택하는 방법은 선두주자 항목을 선택한다음 Shift 키를 누른 채로 끝주자 항목을 선택하면됨.
- 항목을 선택 후 Ctrl + D 키를 누르면 그 항목을 복제할 수 있음.
