1. ABP_Character -> 애님그래프 -> Layered blend per bone 노드 선택 -> 디테일 -> 런타임 -> 웨이트 블렌드 -> 바인드 클릭 -> 핀으로 노출 해제.
- 웨이트 블렌드 옵션은 베이스 포즈와 블렌드 스페이스 포즈가 서로 섞이는 비율임.
- 보통은 두 종류의 애니메이션을 각각 비율을 분배하는 것은 부자연스러운 애니메이션이 출력되기 때문에 이 옵션은 사용을 잘 안함.
- 대신에 스켈레탈 본 영역에서 부위마다 세부적으로 분배해서 더 자연스럽게 출력하는 기능이 있음.

### Depth Blend
- 자연스러운 블렌드를 위해서 점진적으로 블렌드를 수행하는 기능.
#### Branch Filter
예시 1)
- Layered blend per bone 노드 선택 -> 디테일 -> 설정 -> 블렌드 모드 -> Branch Filter.
- 레이어 설정 -> 인덱스 0 -> 분기 필터 -> 인덱스 추가 -> 인덱스 0 -> 본 이름: spine_01, 뎁스(깊이) 블렌드: 0
- SK_Mannequin -> 스켈레톤 트리를 보면 spine_01 은 상체 본 영역의 시작 본임. (허리 -> 팔 -> 머리)
- 깊이가 0이면, spine_01 이하의 본들은 100% 비율로, 블렌드 포즈 핀으로 넣어준 어퍼바디 포즈를 취함.
- 그 외에 본들은 100% 비율로, 베이스 포즈 핀으로 넣어준 풀바디 포즈를 취함.
- 즉, 상체와 하체가 따로 분리되어 서로 다른 애니메이션이 섞이지 않고 순수 100% 동작하게됨. (깊이 1과 같은 취급함.)
- 이번엔 깊이가 3이라고 가정.
- spine_01: 베이스 포즈에 블렌드 포즈가 1/3 만큼 섞여서 포즈를 취함. (깊이 레벨 1)
- spine_02: 베이스 포즈에 블렌드 포즈가 1/2 만큼 섞여서 포즈를 취함. (깊이 레벨 2)
	- 부모 본 내의 spine_02 와 같은 레벨의 본들도 블렌드 포즈가 1/2 섞임.
- spine_03: spine_03 본 부터는 블렌드 포즈가 100%로 포즈를 취함. (깊이 레벨 3)
	- 부모 본 내의 spine_03 과 같은 레벨의 본들도 블렌드 포즈 100% 적용.
- 즉, 블렌드 포즈의 비율을 점점 높혀가며 두 포즈 간의 간극을 좁히고 부드럽게 이어지게함.
예시 2)
- 인덱스 0 -> 본 이름: pelvis (전신의 시작 본, 골반), 뎁스 블렌드: 5 (pelvis 의 하위 계층의 최대 깊이)
- 인덱스 1 -> 본 이름: thigh_l (왼쪽 하반신 시작 본), 뎁스 블렌드: -1 (-1: 이 본(thigh_l) 부터는 블렌드 포즈를 섞지 않음)
- 인덱스 2 -> 본 이름: thigh_r (오른쪽 하반신 시작 본), 뎁스 블렌드: -1 (이 본 부터는 블렌드 포즈를 섞지 않음)
- 골반부터 상반신 전부는 베이스 포즈에 블렌드 포즈를 점진적으로 섞어가며 포즈를 취함.
- 골반 아래 하반신은 베이스 포즈에 블렌드 포즈를 전혀 섞지 않고 순수 100% 베이스 포즈를 취함.
- 즉, pelvis 가 하반신을 포함하는 클래스이기 때문에 하반신은 따로 -1 처리함.
#### Blend Mask
- 좀 더 세부적인 블렌딩 비율을 섞을 수 있는 모드.
예시)
- Layered blend per bone 노드 선택 -> 디테일 -> 설정 -> 블렌드 모드 -> Blend Mask.
- SK_Mannequin -> 스켈레톤 트리 -> 설정 -> 블렌드 프로파일 -> 블렌드 마스크 추가.
- 이름 예시: BlendTest
- 스켈레톤 트리에서 각 본 클래스에 직접 블렌딩 비율을 넣어줄 수 있음.
- Layered blend per bone 노드 선택 -> 디테일 -> 블렌드 마스크 -> 인덱스 0 -> BlendTest 선택하면 끝.
	- 같은 마스크를 여러곳에서 공유할 수 있는 장점이 있음. (유지보수 용이)
