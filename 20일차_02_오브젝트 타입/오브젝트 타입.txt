# 오브젝트 타입
해당 오브젝트 타입의 물체가 어떤 종류의 물체로 취급될 것인지를 정의하는 태그.
1. BP_BoxCollision -> Box 컴포넌트 -> 디테일 -> 콜리전 프리셋 -> 오브젝트 타입.
- WorldStatic: 월드상의 움직이지 않는 물체.
- WorldDynamic: 월드상의 움직이는 물체.
- Pawn: Pawn 타입 물체.
- PhysicsBody: 물리를 적용받는 물체. (피직스 시뮬레이트 옵션을 켠)
- Vehicle: 이동 수단 또는 탑승 수단 물체.
- Destructible: 부서질 수 있는 물체.

# 새로운 오브젝트 타입 추가
- 어떠한 물체는 기존 정의된 오브젝트 타입들로 취급하기 어려울 수 있기 때문에, 새로운 오브젝트 타입을 정의할 수 있음.
1. 프로젝트 세팅 -> 엔진 -> 콜리전 -> ObjectChannels.
- 여기서 새로운 오브젝트 타입을 추가하면 다른 곳에서 이 오브젝트 타입을 선택할 수 있고, 이미 정의된 콜리전 프리셋들에서 이 타입이 추가됨.
- 기본 반응 설정은 예를들어 ignore 로 지정하면 이미 정의된 프리셋들에서 이 오브젝트 타입과의 콜리전 반응이 ignore 로 반영됨.
