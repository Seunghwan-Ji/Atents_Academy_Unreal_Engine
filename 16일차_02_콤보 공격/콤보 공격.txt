1. 어택1~3 몽타주 모두 진입.
2. 1번트랙 -> 노티파이 스테이트 추가 -> 몽타주 노티파이 창.
- 히트 구간을 만들어서 그 구간에만 마우스 클릭 입력을 받아서 알리기 용도.
- 마우스를 실수로 더블클릭해도 콤보가 2타까지 안 나가도록 방지 목적.
3. 노티파이 우클릭 -> Notify Frame: 노티파이 시작 프레임, Anim Notify State Frames: 노티파이 끝 프레임. ()
- 예시
- 어택1 몽타주 -> 15프레임부터 25프레임 동안.
- 어택2 몽타주 -> 20프레임부터 30프레임 동안.
- 어택3 몽타주 -> 20프레임부터 30프레임 동안.
4. BP_Character -> 이벤트 그래프 -> Event BeginActionA 노드부터 로직 작성.
- 수업 시간에 작성한 로직 확인. (콤보 공격 로직)
