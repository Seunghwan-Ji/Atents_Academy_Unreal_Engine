1. 레벨 뷰포트 -> 액터 배치 -> 셰이프 -> 큐브 -> 월드에 드래그 배치.
2. 큐브 액터 선택 -> 디테일 -> 추가 버튼 오른쪽에 블루프린트 클래스 아이콘 클릭.
- 이름 예시: BP_Cube
3. BP_Cube -> 이벤트 그래프에서 수업 시간에 작성한 로직 확인.
4. 충돌에 관여할 수 있는 컴포넌트 종류
- 씬 컴포넌트라고 해서 무조건 충돌에 관여하는건 아님.
- UPrimitiveComponent(기하학적인 형태를 가진 컴포넌트들의 원형)에 속한 컴포넌트들은 다 충돌에 관여하고, Sweep 판정을 할 수 있음.
- 대표적으로 MeshComponent 의 StaticMeshComponent, SkeletalMeshComponent 등이 있고, 이 컴포넌트들은 충돌에 관여함.
5. 레벨 뷰포트 -> 배치된 BP_Cube 의 인스턴스 선택 -> 디테일 -> 피직스 -> 피직스 시뮬레이트 옵션을 체크하면 플레이 시 큐브가 중력을 받아 자유낙하함.
6. 중력을 받아 자유 낙하하는 물리적인 충돌 검사를 위해서는 두 충돌체 모두 콜리전 활성화됨 옵션에서 Collision Enabled(쿼리 앤 피직스) 옵션이 선택되어야함.
- 예시 상황: BP_BoxCollision 충돌체 위에서 떨어지는 BP_Cube 충돌체. (레벨에서 위 아래로 배치)
7. BP_BoxCollision -> Box -> 디테일 -> 콜리전 -> 시뮬레이션 중 히트 이벤트 생성 체크.
- BP_BoxCollision 이 피직스 시뮬레이션 중이거나, 피직스 시뮬레이션 중인 충돌체(BP_Cube)가 BP_BoxCollision 과 충돌했을 때 BP_BoxCollision 에서 히트 이벤트를 발생시킬 지 여부.
