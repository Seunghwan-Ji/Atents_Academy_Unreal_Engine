1. BP_Character -> 컴포넌트 -> 스켈레탈 메시 -> 디테일 -> 메시 -> 스켈레탈 메시 에셋(SKM_Mannequin) 더블 클릭 -> 우측 상단 관절 모양 버튼(PA_Mannequin) 클릭.
- 스켈레탈 메시의 각 관절의 움직임 범위를 지정해 놓은 에셋.
- 뷰포트 바깥 상단 ">>" 버튼 -> 시뮬레이트를 누르면 캐릭터가 쓰러졌을 때 관절이 어떻게 움직이는지 볼 수 있음.
- 보라색 영역은 각 본 클래스의 충돌 영역임.
	- BP_Character -> 컴포넌트 -> 스켈레탈 메시 -> 디테일에서 설정한 콜리전 항목이 이 영역들과 충돌 했을 때 반영됨.
	- 보라색 영역 클릭 -> 디테일 -> 콜리전 항목에서 그 영역만 다른 충돌 처리를 반영하게 할 수 있음.
