# 피지컬 머티리얼
- 충돌, 마찰, 반발, 소리, 파티클 효과 등 물리적 반응에 영향을 주는 속성을 정의한 에셋 (표면에 대한 물리 정보)
1. 콘텐츠 드로어 -> 추가 -> 피직스 -> 피지컬 머티리얼.
- 이름 예시: PM_Example
2. PM_Example -> 디테일.
- 마찰: 마찰 계수.
- 스태틱 마찰: 정지 마찰 계수.
- 복원력: 탄성력.
- 피지컬 프로퍼티 -> 표면 타입은 예를들어 표면 타입별로 마찰했을 때 사운드를 다르게 하는데 쓰임. (예: 나뭇잎 마찰음, 마루 바닥 마찰음 등)

# 테스트
1. BP_Cube -> 스태틱 메시 컴포넌트 -> 디테일 -> 콜리전 -> 피직스 머티리얼 오버라이드 -> PM_Example 선택.
2. PM_Example -> 디테일 -> 복원력: 0.3 -> 1
- 플레이해서 복원력 0.3일 때와 1일 때 큐브의 탄성력 체감해보기.
